# phishing-detection-extension
 Phishing Detection Browser Extension  is a lightweight and intelligent browser extension designed to protect users from phishing attacks in real-time. It analyzes URLs, webpage content, and security indicators to detect and block malicious websites before they can compromise sensitive information.
 
 🚀 Features
🛡️ Real-Time URL Scanning

Detects suspicious and malicious URLs instantly when a user visits a website.
🔍 Machine Learning Detection
Uses trained models to identify phishing patterns based on URL structure and page behavior.
⚠️ Warning Alerts
Displays instant alerts when a phishing site is detected.
🌐 Blacklist & Whitelist Support
Maintains a database of known phishing domains and trusted websites.
🔒 SSL Certificate Verification
Checks HTTPS and certificate validity for additional security.
📊 Heuristic Analysis
Identifies phishing attempts using rules like:
Suspicious domain names
URL shortening services
Excessive subdomains
Fake login forms
⚡ Lightweight & Fast
Minimal performance impact on browsing.
🧠 How It Works
User visits a website
Extension scans the URL and webpage
Runs checks using:
Blacklist database
Heuristic rules
ML model (if enabled)
If suspicious → shows warning / blocks access
🛠️ Tech Stack
JavaScript (Vanilla / React optional)
HTML & CSS
Chrome Extension APIs
Python (for ML model – optional)
TensorFlow / Scikit-learn (if ML integrated)
